## 🥪 Install
```shell
$ npm install @muxed/rpg-utils
```

## 🎉 Usage

Import the instaled Package
```ts
import { Dice, Rolling } from '@muxed/rpg-utils'
```

## 🟢 Testing
Install the Jest dependencies if not in global
```shell
$ npm install
```
Run the tests
```shell
$ npm test
```
